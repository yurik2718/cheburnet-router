cheburnet-router

Образовательный OpenWrt-стенд — AmneziaWG, split-routing, DoH, kill switch.
О проекте. Это демонстрация современных сетевых технологий. VPN-туннели (AmneziaWG/WireGuard) — стандартная индустриальная технология: применяются в корпоративных сетях, банкинге, межсерверной связи по всему миру. Здесь мы настраиваем их на домашнем роутере для учебных целей.

Шаг 1 из 3 · Файл AmneziaWG-конфига

Загрузите .conf-файл от вашего AmneziaWG-сервера. В приложении Amnezia: Настройки → Сервер → Поделиться → Экспорт конфигурации.

Нет своего сервера? Самый простой вариант — Amnezia Premium (готовая подписка, экспорт .conf в один клик). Реферальная ссылка — поддерживает развитие проекта.

Шаг 2 из 3 · Wi-Fi

Все устройства, подключённые к этой Wi-Fi-сети, автоматически получат настройки — ничего не нужно настраивать на каждом телефоне/ноутбуке.

Шаг 3 из 3 · Подтверждение

Установка займёт ~12 минут. Роутер скачает и настроит AmneziaWG, sing-box, adblock-lean, DNS и остальные компоненты. Вы будете видеть прогресс в реальном времени.
