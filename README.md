# ПОЧЕМУ ЭТО ЛУЧШАЯ СБОРКА — ЧЕСТНО

## САМАЯ ПРОДВИНУТАЯ ОБФУСКАЦИЯ ИЗ ДОСТУПНЫХ

Обычный WireGuard блокируется российскими провайдерами по сигнатуре — это факт. Эта сборка использует **AmneziaWG 2.0 + I1 CPS**: туннель генерирует decoy-пакеты, неотличимые от обычного HTTPS-трафика. Провайдер видит «что-то зашифрованное» — не VPN, не WireGuard, ничего конкретного. Это не маркетинг — это конкретный протокол с открытым кодом, который можно разобрать по байтам: [docs/02-amneziawg.md](docs/02-amneziawg.md).

## ТРЁХСЛОЙНАЯ ЗАЩИТА ОТ УТЕЧЕК — КАК В КОРПОРАТИВНЫХ СЕТЯХ

Большинство VPN-решений дают один слой защиты: упал VPN — трафик ушёл напрямую. Здесь три независимых слоя: **sing-box bind-outbound** блокирует трафик на уровне прокси, **TPROXY** блокирует на уровне ядра, **fw4 firewall rules** блокирует на уровне межсетевого экрана. Чтобы трафик утёк, нужно чтобы все три одновременно отказали — это практически невозможно. Подробно: [docs/08-killswitch.md](docs/08-killswitch.md).

## УМНАЯ МАРШРУТИЗАЦИЯ БЕЗ ПОТЕРИ СКОРОСТИ

`.ru`, `.su`, `.рф`, Госуслуги, VK, Сбер — идут напрямую с полной скоростью провайдера. YouTube, Instagram, Telegram — через VPN-туннель. Это не «список сайтов который нужно обновлять вручную» — **Podkop + sing-box FakeIP** автоматически определяет маршрут по TLD и живым спискам доменов. Настроил один раз — работает само.

## БЕСПЛАТНЫЙ РЕЖИМ БЕЗ VPN-СЕРВЕРА

Если платить за VPN не хочется — есть режим **zapret**: роутер разбивает TLS-хэндшейк так, что DPI-оборудование не распознаёт сайт. YouTube, Twitch — без оплаты, без внешнего сервера, полностью локально. Это не «заглушка» — это отдельный полноценный режим с той же установкой через браузер.

## ТО ЖЕ ЖЕЛЕЗО, В ТРИ РАЗА ДЕШЕВЛЕ

GL.iNet Beryl AX стоит $110–140 и считается эталоном travel-роутера. **Cudy TR3000 стоит $40–55 и построен на том же чипе** — MediaTek MT7981B, те же 512 МБ RAM, тот же Wi-Fi 6. Весь функционал этой сборки работает одинаково на обоих. Разница — только физический слайдер (есть у Beryl AX) и цена.

## ЧЕСТНО: ЧТО ЗДЕСЬ НЕТ

- Бесплатного «всё включено» нет — для VPN-режима нужен платный сервер (~300–500 руб/мес)
- Роутер нужно прошить на OpenWrt — это 30 минут по инструкции, один раз
- Это не защита от АНБ и не анонимность — это приватность и обход блокировок

## Быстрый старт — веб-мастер

Не нужны git, bash, Linux — только браузер. Один раз через SSH запускаем веб-мастер:

```bash
ssh root@192.168.1.1 'wget -qO- https://raw.githubusercontent.com/yurik2718/cheburnet-router/main/bootstrap.sh | sh'
```

Открываем в браузере:

```
http://192.168.1.1/cheburnet/
```

Веб-мастер объяснит разницу между VPN и zapret, примет `.conf`-файл, настроит Wi-Fi и сделает всё остальное. Прогресс — прямо в браузере. После установки там же — управление и статус.

**Альтернатива для продвинутых** (Linux/macOS):

```bash
git clone https://github.com/yurik2718/cheburnet-router.git
cd cheburnet-router
./setup.sh
```

## Два режима — какой выбрать?

### VPN (AmneziaWG) — рекомендуем

Трафик идёт через зашифрованный туннель до VPN-сервера за рубежом. Провайдер видит только зашифрованный поток — ни сайты, ни содержимое.

**Открывает:** всё — YouTube, Instagram, Telegram, любые IP-блокировки.  
**Требует:** платный VPN (~300–500 руб/мес) — Amnezia VPN или свой VPS.

### zapret — бесплатно, без сервера

Роутер «ломает» TLS-хэндшейк так, что DPI-оборудование провайдера не распознаёт сайт. Работает локально, без внешнего сервера.

**Открывает:** YouTube, Twitch и сайты с DPI-блокировкой.  
**Не открывает:** Instagram, Facebook — у них IP-блокировка, не DPI.  
**Не скрывает:** провайдер видит, какие сайты вы открываете.

> Если нужен Instagram или важна приватность — берите VPN. zapret — лучшее из бесплатного.

## Что происходит после установки

| Трафик | Куда идёт |
|---|---|
| yandex.ru, vk.com, gosuslugi.ru | Напрямую — без VPN, полная скорость |
| youtube.com, instagram.com, telegram.org | Через зашифрованный VPN-туннель |
| Реклама и трекеры | Заблокированы на уровне DNS |

Переключение режима — одна команда:

```bash
ssh root@192.168.1.1
vpn-mode home    # .ru напрямую + остальное через VPN (по умолчанию)
vpn-mode travel  # весь трафик через VPN
vpn-mode status  # текущий режим
```

## Что установлено и как работает

### Режим VPN

| Компонент | Что делает |
|---|---|
| **AmneziaWG 2.0 + I1 CPS** | VPN с обфускацией — выглядит как обычный HTTPS-трафик |
| **Podkop + sing-box** | Split-routing: .ru/.su/.рф напрямую, всё остальное через VPN |
| **adblock-lean + Hagezi Pro** | Блокировка 200k+ рекламных доменов на уровне DNS |
| **Quad9 DoH** | Зашифрованный DNS — провайдер не видит ваши запросы |
| **Kill switch (3 слоя)** | При падении VPN трафик блокируется — ничего не утекает |
| **AWG watchdog** | Авто-перезапуск туннеля при зависании handshake |
| **WPA3** | Современное шифрование Wi-Fi |

### Режим zapret

| Компонент | Что делает |
|---|---|
| **zapret / nfqws** | Перехватывает пакеты через nfqueue, разбивает TLS-хэндшейк — DPI не распознаёт сайт |
| **adblock-lean + Hagezi Pro** | Блокировка рекламы и трекеров |
| **Quad9 DoH** | Зашифрованный DNS |
| **WPA3** | Современное шифрование Wi-Fi |

## Для дома — настроил один раз, работает годами

- **Kill switch всегда активен** — сбой VPN не приводит к утечкам
- **AWG watchdog** — туннель автоматически восстанавливается при зависании
- **Quad9 DoH** — зашифрованный DNS без дополнительной настройки
- **adblock-lean** защищает все устройства в сети — телефоны, Smart TV, планшеты

## Для путешественников — полный travel-набор

Компактный **Cudy TR3000** работает от PowerBank 5V. Подключили в отеле — все ваши устройства в защищённой сети. Отель видит одного клиента — ваш роутер.

- **`travel-connect`** — подключение в WISP-режиме (роутер как Wi-Fi клиент)
- **`travel-portal`** — обход captive portal с авто-восстановлением через N минут
- **`travel-check`** — полная диагностика одной командой
- **`travel-vpn-on`** — ручное восстановление VPN после портала

## Пошаговый гайд

### Шаг 1 — Купить роутер

**Рекомендуется: Cudy TR3000 v1** (~$40–55 на AliExpress)

- Компактный, питание от USB-C (работает от повербанка)
- Wi-Fi 6, 2.5 GbE, 512 МБ RAM
- Тот же чип (MT7981B), что у GL.iNet Beryl AX — в 2–3 раза дешевле

> ⚠️ **Серийники `2543...`** (ноябрь 2025+) — нужна OpenWrt 24.10.5+, иначе кирпич.  
> ❌ **Не берите** MT7621-роутеры (WR1300, M1300, X6) — 128 МБ RAM, не хватит.

→ [Таблица совместимых моделей ↓](#совместимое-железо)

---

### Шаг 2 — Прошить OpenWrt

1. Зайдите на **[cudy.com/openwrt-software-download](https://www.cudy.com/en-us/blogs/faq/openwrt-software-download)**
2. Скачайте **interim firmware** → прошейте через веб-UI Cudy (Firmware Upgrade)
3. После перезагрузки скачайте **OpenWrt 25.12.2** с [openwrt.org/toh](https://openwrt.org/toh/start) → прошейте снова
4. Проверьте SSH: `ssh root@192.168.1.1` (по умолчанию пустой пароль)

> Подробности и нюансы прошивки: [docs/10-upgrades.md](docs/10-upgrades.md)

---

### Шаг 3 — Получить AmneziaWG конфиг (для режима VPN)

**Вариант А — Amnezia VPN** (платная подписка, самый простой):
> Купите подписку на [amnezia.org](https://amnezia.org). В приложении: Настройки → Сервер → Поделиться → Экспорт конфигурации → скачайте `.conf`

**Вариант Б — Свой VPS с AmneziaWG**:
> Арендуйте VPS (~$3–5/мес — Hetzner, DigitalOcean, Timeweb).  
> Установите через приложение Amnezia → инструкция за ~5 минут. Экспортируйте `.conf`.

> Как работает AmneziaWG изнутри: [docs/02-amneziawg.md](docs/02-amneziawg.md)

```bash
# Один раз через SSH:
```

### Шаг 5 — Проверить работу

1. Подключитесь к Wi-Fi с вашим новым SSID
2. Откройте **[speedtest.yandex.ru](https://speedtest.yandex.ru)** — должен работать напрямую
3. Откройте **[speedtest.net](https://speedtest.net)** — в России заблокирован, значит откроется только через VPN

```bash
ssh root@192.168.1.1 'travel-check'   # полная диагностика одной командой
```

## Совместимое железо

### Рекомендованные роутеры

| Модель | Цена | Особенности |
|---|---|---|
| **Cudy TR3000 v1** ⭐ | ~$40–55 | Travel-форм-фактор, USB-C 5V (PowerBank), 2.5 GbE |
| Cudy WR3000P v1 | ~$50–65 | 4×GbE LAN + 2.5 GbE WAN, стационарный |
| Cudy AP3000 v1 | ~$60–75 | 256 МБ flash, больше запаса overlay |
| GL.iNet Beryl AX | ~$110–140 | Физический слайдер HOME/TRAVEL + активное охлаждение |

**Все модели** используют MediaTek MT7981 (Filogic) — пакеты AmneziaWG совместимы без изменений.

### Cudy TR3000 vs GL.iNet Beryl AX — честное сравнение

Железо почти идентичное — оба на MediaTek MT7981B:

| | GL.iNet Beryl AX | Cudy TR3000 |
|---|:---:|:---:|
| SoC | MT7981B | **тот же** |
| RAM | 512 МБ | 512 МБ |
| Flash | 256 МБ | 128 МБ |
| Wi-Fi | 6 AX3000 | 6 AX3000 |
| Активное охлаждение | Есть (кулер) | Нет (пассивное) |
| Физический слайдер HOME/TRAVEL | ✅ | ❌ (CLI `vpn-mode`) |
| **Цена** | **~$110–140** | **~$40–55** |

**О flash:** наша сборка использует ~27 МБ. На Cudy (128 МБ) остаётся ~100 МБ overlay — с большим запасом.  
**О кулере:** trip point термозоны — 60°C. В домашнем режиме CPU редко превышает 55–58°C. Кулер почти не включается даже на Beryl AX.

**Вывод:** если нужен физический переключатель для нетехнических пользователей в семье — Beryl AX. Если хотите тот же функционал за треть цены — Cudy TR3000.

### Минимальные требования

- ≥ 256 МБ RAM (рекомендуется 512 МБ)
- ≥ 64 МБ flash
- OpenWrt 23.05+

> Портирование на другое железо: [setup/README.md](setup/README.md)

## Честные ограничения

- ❌ Не защищает от физического доступа к роутеру
- ❌ Не гарантирует невидимость при продвинутом статистическом DPI
- ❌ Не блокирует 100% рекламы (YouTube-реклама идёт с тех же серверов что и видео)
- ❌ Не лечит malware на устройствах в сети
- ❌ Нужен платный VPN-сервер или свой VPS для режима VPN

## Дисклеймер

**Технологии VPN** (AmneziaWG, WireGuard, OpenVPN) **легальны** в большинстве стран. Они стандартно применяются в корпоративных сетях, для удалённой работы, защиты трафика в публичных Wi-Fi и приватности. Этот проект создан в образовательных целях. Автор не несёт ответственности за применение описанных технологий читателями.
